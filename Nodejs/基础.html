# 异步 IO

  1. IO 是应用程序的瓶颈所在
  2. 异步 IO 提高性能无采原地等待返回结果
  3. IO 操作属于操作系统级别，平台都有对应实现
  4. Nodejs 单线程配合事件驱动架构及 libuv 实现了异步 IO
